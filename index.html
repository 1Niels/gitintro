git init

git init [project-name]

git init creates an empty local Git repository or reinitializes an existing one with the specified name.

git branch

git branch [branch-name]

git branch lists, creates, or deletes branches. To list all local branches in the current repository do not include a branch name

git checkout

git checkout [branch-name]

git checkout switches to the specified branch and updates the working directory

git status

git status

git status shows the working tree status, it lists all new modified files to be committed

git log

git log

Lists verision history for the current branch, to list version history for a file, including renames type
git log --follow [file]

git push

git push [alias] [branch]

git push updates all local branch commits to GitHub

  • Copy

git commit

git commit -m "[message describing changes]"

git commit records changes to the repository permanently in version history.

  • Learn more

git rebase

git rebase [branch]

git rebase forward-ports local commits to the updated upstream head, takes all changes that were committed on one branch and replays them on another one.

  • Learn more

git stash

git stash

git stash temporarily stores all modified tracked files.

  • Learn more

git reset

git reset [commit]

git reset undoes all commits after [commit], it preserves changes locally.

  • Learn more

git add

git add [file]

Snapshots the file in preparation for verioning. To add all files to index type
git add .

git diff

git diff [filename]

git diff checks differences between the working directory and the staging area.
